# Mini Project: 2D Shooting Game with Phaser

This repository contains a self-learning mini project focused on basic 2D game development using **Phaser.js**.  
The project was created to practice JavaScript programming, game logic, and asset management.

---

## Project Structure

Inside the `miniproject` folder, the structure is organized as follows:

```
miniproject/
│
├── assets/
│   └── (image files used in the game)
│
├── Game.js
├── indexza.html
└── phaser.min.js
```

---

## File Description

- **assets/**  
  A folder used to store all image assets used in the game, such as characters, objects, and backgrounds.

- **Game.js**  
  The main game file that contains the core game logic, including player movement and gameplay mechanics.

- **indexza.html**  
  The main HTML file used to load and run the game in a web browser.

- **phaser.min.js**  
  The Phaser.js library file used as the game engine for developing this game.

## Purpose
This mini project was developed for self-learning purposes to gain hands-on experience in:
- Basic 2D game development
- JavaScript game logic
